POC Demo Retailer: —

0 API Token

Paste your JWT bearer token (generate with python -c "from kistbook.core.security import create_access_token; print(create_access_token({'sub':'demo'}))")

1 Create Retailer

Register ShopHive as a retailer. The returned ID is saved automatically for subsequent steps.

2 Upload Installment Book CSV

Upload a CSV with columns: customer_name, cnic, phone, guarantor_name, guarantor_phone, installment_amount, due_day_of_month, total_installments, installments_paid, last_payment_date

3 Trigger Daily Scan

Runs the reminder engine scan immediately. Only available in development environment.

4 View Reminder Logs

Audit trail of all outbound reminders and inbound replies.
